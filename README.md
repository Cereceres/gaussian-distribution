# gaussian-distribution
## Install

```bash
$ npm install gaussian-pdf
```
## Test

```js
let pdf = require('gaussian-pdf')
pdf() // -> -1.12565465
```
## API  pdf([mu = 0, sigma = 1])

The function receive the mu and sigma params to normal distribution, default values are 
mu = 0 and sigma = 1.
